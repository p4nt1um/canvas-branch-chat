# v0.2.0 Release Notes

## 🎬 P2 #14: 对话回放

右键任意节点 → **▶️ 回放对话**，三种遍历模式逐节点聚焦回顾：

- **T 时间线**：按节点创建时间顺序（`createdAt` 时间戳）
- **B 广度优先**：BFS 层序，同层按创建时间排序
- **D 深度优先**：沿分支走到底再回溯

### 控制

- ⏯ **空格** 暂停/继续
- ⏮⏭ **←/→** 上一个/下一个
- **Esc** 退出回放
- **T/B/D** 实时切换遍历模式
- 三档速度：慢 / 中 / 快
- 控制条浮动在屏幕顶部，不遮挡画布

### 视觉

- 全局总览 → 逐节点 `zoomToBbox` 聚焦（Obsidian 原生动画）
- 每 4 张回一次总览锚定方位
- pending 节点灰暗，played 节点边框标记，current 节点高亮

## ✨ P2 #22: 智能追问改进

- **分级降级提取**：标题(#~####) → 数字列表 → bullet → 第1行
- 上级 ≥3 个时只取上级，<3 时自动降级
- 全部不足时阈值降到 2 再走一遍，仍不满足取第1行语句
- 正则提取不足 2 个时，fallback 到 LLM 生成候选问题

## 📝 总结流程改进

- 多选合并时，**先创建 user 节点显示提示词**，再创建 assistant 节点显示总结
- 连线：源节点 → user(提示词) → assistant(总结)
- 与追问流程保持一致，提示词不再"隐身"

## 🔧 其他

- 节点新增 `createdAt` 时间戳字段，写入 `Date.now()`
- 时间线排序使用真实创建时间，旧节点 fallback 到坐标排序
- 回放全部使用 Obsidian Canvas 原生 API（`zoomToFit`/`zoomToBbox`），不自建 viewport 动画
- 控制条按钮使用 `pointerdown` 事件，避免 Canvas 事件捕获冲突
- 速度下拉菜单方向修正（向下展开）
